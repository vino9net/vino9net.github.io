# vino9.net

Static site built with [Astro](https://astro.build) using a port of the
[Hugo Terminal](https://github.com/panr/hugo-theme-terminal) theme by
[Dennis Klappe](https://github.com/dennisklappe/astro-theme-terminal).

## Develop

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # outputs to ./dist
npm run preview
```
